# Usage

1.
```js
yarn
yarn start
```

2.
Change `android/app/src/main/res/values/strings.xml` to your appID/
https://developers.facebook.com/apps
```
<resources>
    <string name="app_name">Project</string>
    <string name="facebook_app_id">Your FB APP ID </string>
</resources>
```
